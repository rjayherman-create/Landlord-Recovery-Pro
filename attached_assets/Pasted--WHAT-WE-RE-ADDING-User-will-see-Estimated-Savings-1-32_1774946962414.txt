🧠 HOW IT WORKS (SIMPLE + EFFECTIVE)

We calculate based on:

1. Over-assessment gap  
2. Number of comps  
3. Strength of comps vs assessment  
🚀 STEP 1 — CREATE CALC FUNCTION

In frontend (or backend if you prefer):

const calculateConfidence = (marketValue: number, assessedValue: number, comps: any[]) => {
  if (!marketValue || !assessedValue) return { score: 0, label: "Low" };

  const gap = assessedValue - marketValue;

  // % over-assessment
  const percentDiff = gap / marketValue;

  let score = 0;

  if (percentDiff > 0.2) score += 40;
  else if (percentDiff > 0.1) score += 25;
  else if (percentDiff > 0.05) score += 10;

  // comps strength
  if (comps.length >= 3) score += 30;
  else if (comps.length >= 1) score += 15;

  // base score
  score += 20;

  // cap at 95%
  score = Math.min(score, 95);

  let label = "Low";

  if (score > 70) label = "High";
  else if (score > 40) label = "Moderate";

  return { score, label };
};
🚀 STEP 2 — STORE RESULT
const [confidence, setConfidence] = useState({ score: 0, label: "Low" });
🚀 STEP 3 — UPDATE WHEN DATA CHANGES
useEffect(() => {
  const result = calculateConfidence(
    Number(marketValue),
    Number(assessedValue),
    comps
  );

  setConfidence(result);
}, [marketValue, assessedValue, comps]);
